# RoadWatch Monitor
Система мониторинга состояния дорожного полотна с интеграцией метеоданных.

## Технологии
* Python / Django
* React (Frontend)
* Leaflet (Картография)
* Open-Meteo API (Погода)

## Как запустить
1. Установить зависимости: `pip install -r requirements.txt`
2. Запустить сервер: `python manage.py runserver`
